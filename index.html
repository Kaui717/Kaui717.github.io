Git

1)查看版本号 git –version

2)设置用户签名（与代码托管平台的账号无关，这个只是代表了windows的本地客户端） 首次用Git一定要设置用户签名，否则提交代码的时候会报错。

3)初始化本地库 为的是获得目录的管理权 git init

4)查看仓库状态 git status

5)添加暂存区 git add 文件名

6)提交本地库 git comit -m “日志信息” 文件名

7)查看简要历史版本 git reflog

查看详细历史版本 git log

8)版本穿梭 git reset –hard 版本号 版本号改变之后工作区的文件也会相应的改变。

9)分支创建 git branch 分支名

​ 分支查看 git branch -v

​ 分支切换 git checkout 分支名

​ 合并分支 git merge 分支名

Github

1)测试账号：Kaui717

2)查看别名 git remote -v

​ 创建别名 git remote add 别名 远程地址

3)推送本地分支到远程库 git push 远程别名 分支名

4)克隆 git clone 远程地址 自动拉取代码 初始化本地库 创建别名

5)推送 git push 远程地址 分支名

6)团队内协作 跨团队协作

7)ssh免密登录

Hexo

1)安装Hexo的时候会安装Hexo和一些命令

2)初始化一个站点 hexo init 站点名 同时生成一些文件，这个文件是通过git拉取的。

3)进入站点 npm install

4)生成网站页面 hexo g 生成一个public文件夹

5)运行网站 hexo s

6)写作 hexo new “文章标题”

生成草稿 hexo new draft “标题”

生成此纯页面 hexo new page “标题”
